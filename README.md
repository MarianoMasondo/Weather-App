# Weather App

Aplicación meteorológica desarrollada con **React**, **Vite** y **Material UI**.  
Permite consultar el clima actual de una ciudad y también intenta detectar la ubicación del usuario para mostrar el clima automáticamente.

## Demo

🔗 [Ver aplicación deployada]((https://marian-weather.vercel.app/))

## Descripción

Weather App es una aplicación web que consume una API meteorológica para mostrar información actual del clima. Al iniciar, la app intenta obtener la ubicación del usuario mediante el navegador. Si el usuario no permite el acceso a la ubicación, se puede buscar manualmente cualquier ciudad desde el formulario de búsqueda.

La aplicación muestra datos como:

- Ciudad consultada
- Temperatura actual
- Sensación térmica
- Humedad
- Velocidad del viento
- Condición climática
- Ícono del clima
- Fecha
- Hora de salida del sol
- Hora de puesta del sol

## Tecnologías utilizadas

- React
- Vite
- JavaScript
- Material UI
- HTML
- CSS
- World Weather Online API
- Vercel

## Funcionalidades principales

- Búsqueda de clima por ciudad.
- Obtención automática del clima según la ubicación del usuario.
- Manejo de errores cuando el campo de ciudad está vacío.
- Visualización de información climática en una tarjeta.
- Diseño responsive con Material UI.
- Deploy en Vercel.
